# Canadian Policyholder Portal

Monorepo with:

- `client/`: React (Vite) frontend (port 3000)
- `server/`: Express.js backend API (port 5000)

## Prereqs

- Node.js 18+

## Setup

```bash
npm install
```

## Run (dev)

### Both (recommended)

```bash
npm run dev
```

### Server

```bash
npm -w server run dev
```

### Client

```bash
npm -w client run dev
```

## Lint / Test / Build

```bash
npm run lint
npm run test
npm run build
```
